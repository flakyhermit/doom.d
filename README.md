
# My Doom Emacs configuration

## How to set it up
1. Install Emacs (>27)
2. Clone `doom` and install it: 
3. Clone this repository
4. Install soft dependencies
5. Run `doom sync`
6. Setup Org folder

## Soft dependencies
### Programs
- `sqlite3` for `org-roam`.
- `wordnet-cli` and `wordnet-common` for dictionary lookup.
- `fd` for faster searches.
